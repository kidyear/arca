## 一、产品定名(已决策,2026-06-12)

**灵匣 / Arca**——"文件夹操作 + Agent 结合体"的字面翻译:
- **匣** = 文件的容身处(文件匣)——文件管理这半边
- **灵** = 匣中有灵,会自己整理、替你办事——Agent 这半边
- 叙事:上游叫"翻箱"(人翻箱子),我们叫"灵匣"(匣子自己会干活),同宗进化
- 命名权重(用户定):6 分功能、2 分公司品牌(信步 SEAVO)、2 分文化底蕴
- **英文名 Arca**(2026-06-12 定,用户否决拼音 LingXia):拉丁语"匣/柜",与 Ark(约柜,"匣中有灵"的西方原型)同源;品牌区写法 SEAVO ARCA
- 可用标语:匣中有灵 / 文件匣,自己会干活
- 否决过的方向(别再提):拼音直转 LingXia、南桥/板桥(行业件名)、板载/点亮/常亮(黑话)、信手/闲庭/则灵(套用品牌文脉)、干将/搭手/办妥(帮手拟人)、总装/百工台/工坞(泛工作台)
- 候选过的英文名:Emberbox/Sparkcase/Tinderbox(用户选了 Arca)
